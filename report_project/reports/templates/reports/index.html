Сервис отчетов

Добро пожаловать в сервис формирования отчетов! На этой странице рассказано, как использовать две основные вкладки сервиса: "Загрузка файла данных" и "Получение отчета".

Загрузка файла данных

Вкладка "Загрузка файла данных" предназначена для загрузки Excel файлов в базу данных. Для этого выполните следующие шаги:

  1. Перейдите на вкладку "Загрузка файла данных".
  2. Нажмите кнопку "Выбрать файл" и выберите нужный Excel файл на вашем компьютере.
  3. После выбора файла нажмите кнопку "Загрузить", чтобы отправить файл в базу данных.

Получение отчета

На странице отчетов вы можете получить сводную информацию по загруженным отчетам. Для этого выполните следующие шаги:

  1. Перейдите на страницу "Получение отчета".
  2. Просмотрите сводную информацию, представленную на странице.
